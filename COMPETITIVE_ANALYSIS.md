## 1. Профиль конкурента: ИАС «Буратино»

### 1.1 Общее описание

ИАС «Буратино» (Информационно-Аналитическая Система) — коммерческая российская OSINT-платформа, ориентированная на проверку физических лиц. Используется как коммерческими структурами (службы безопасности, HR), так и правоохранительными органами. Закрытый код, веб-интерфейс.

### 1.2 Ключевые входные параметры

| Параметр | Буратино | Примечание |
|----------|----------|------------|
| ФИО | Да | Основной |
| Дата рождения | Да | Основной |
| **Телефон** | **Да (#1 фича)** | Телеком-данные: телефон -> ФИО, адрес |
| ИНН | Да | ЕГРЮЛ/ЕГРИП |
| Паспорт | Да | МВД валидация |
| Госномер авто | Да | ГИБДД |
| Фото | Да | Распознавание лиц |
| Email | Да | Перекрестный поиск |
| Адрес | Да | Росреестр |
| Ссылка на соцсеть | Да | VK, OK, TG |

### 1.3 Источники данных

| Категория | Источники | Доступ |
|-----------|-----------|--------|
| Госреестры | ЕГРЮЛ, ФССП, суды, Росреестр, ГИБДД, МВД, ЕФРСБ, Росфинмониторинг, ФНС | API + партнерства |
| Соцсети | VK (полный), OK.ru, Telegram, Instagram, Facebook | API + скрейпинг |
| Коммерческие | Телеком-данные, утечки, кредитная история, адресные базы | Платные операторы |
| Технологии | Распознавание лиц, обход CAPTCHA, автоматический cross-referencing | Собственная разработка |

### 1.4 Характеристики

| Параметр | Значение |
|----------|----------|
| Скорость | 1-5 минут |
| Стоимость | 3,000-50,000 руб./мес. или 100-1,000 руб./запрос |
| Доступ | Коммерческий + силовые структуры |
| Код | Закрытый |
| Развертывание | Облачное (SaaS) |

### 1.5 Конкурентные преимущества Буратино

1. **Телефон -> ФИО** — главная фича, данные от телеком-операторов (недоступны публично)
2. **Паспорт -> МВД** — проверка действительности паспорта
3. **Росреестр** — собственность на недвижимость по ФИО
4. **ГИБДД** — транспортные средства по госномеру/VIN
5. **Адресная база** — прописка/регистрация
6. **Обход CAPTCHA** — партнерские токены и anti-captcha
7. **Скорость** — 1-5 минут на полную проверку
8. **Единый интерфейс** — все данные в одном окне

### 1.6 Экосистема российских OSINT-инструментов

| Уровень | Примеры | Цена | Аудитория |
|---------|---------|------|-----------|
| **Tier 1: Профессиональные** | Буратино, Cronos, Lampyre | $100-500/год | СБ, правоохрана |
| **Tier 2: Telegram-боты** | Глаз Бога, Quick OSINT, Himera | 50-100 руб./запрос | Массовый рынок |
| **Tier 3: Бесплатные** | Sherlock, Maigret, Snoop, Holehe | Бесплатно | Энтузиасты, OSINT-специалисты |
| **Tier 4: Серый рынок** | Пробив через инсайдеров | 500-5,000 руб./запрос | Нелегальный |

**Позиционирование СЛЕД**: между Tier 1 и Tier 3 — функциональность профессиональной платформы, стоимость бесплатного инструмента.

## 2. Текущее состояние СЛЕД

### 2.1 Входные параметры

| Параметр | Статус | Примечание |
|----------|--------|------------|
| ИНН | ✅ Обязательный | Валидация контрольной суммы (10/12 цифр) |
| ФИО | ✅ Обязательный | Confirmed_name из ЕГРЮЛ |
| Дата рождения | ✅ Опциональный | Фильтрация VK-профилей |
| Город | ✅ Опциональный | Сужение поиска |
| Телефон | ❌ Отсутствует | Нет поля ввода, нет обратного поиска |
| Email | ❌ Отсутствует | Нет поля ввода |
| Паспорт | ❌ Отсутствует | Нет поля ввода и проверки |
| Фото | ❌ Отсутствует | Нет upload UI (Search4Faces интегрирован) |
| Госномер авто | ❌ Отсутствует | Нет функционала |
| Ссылка на соцсеть | ❌ Отсутствует | Нет прямого ввода URL |

### 2.2 Оценка по этапам пайплайна

| Этап | Название | Статус | Что работает | Что не работает |
|------|----------|--------|--------------|-----------------|
| 0 | Идентификация | ✅ Работает | ЕГРЮЛ по ИНН, банкротство, бизнес-сеть | — |
| 1 | Госреестры | ⚠️ Частично | ЕГРЮЛ по имени, суды (sudact.ru), casebook.ru | ФССП (CAPTCHA), checko.ru (404) |
| 2 | Безопасность | ✅ Работает | OpenSanctions, локальные БД МВД/экстремизм, Интерпол | Росфинмониторинг (нужен российский IP) |
| 3 | Соцсети | ⚠️ Частично | VK (с токеном), TG методы A и B | TG метод C (нужна сессия) |
| 4 | Контакты | ⚠️ Частично | 11-шаговая цепочка, утечки, oracle (4/8 сервисов) | Holehe (медленный), oracle (4 сервиса за рубежом) |
| 5 | Глубокий анализ | ⚠️ Частично | YaSeeker | Search4Faces ($40/мес.), Snoop/Maigret (не установлены) |
| 6 | Поведение | ⚠️ Частично | Алгоритмы готовы | Нужен VK_USER_TOKEN для данных стен/друзей |
| 7 | Скоринг | ✅ Работает | 9 категорий + identity flags | — |
| 8 | Отчет | ✅ Работает | HTML, PDF (Playwright/reportlab), JSON | — |

### 2.3 Сводка работоспособности

| Категория | Количество | Детали |
|-----------|------------|--------|
| ✅ Полностью работает | 4 этапа | Stage 0, 2, 7, 8 |
| ⚠️ Частично работает | 5 этапов | Stage 1, 3, 4, 5, 6 |
| ❌ Не работает | 0 этапов | — |
| Общая работоспособность | ~60-65% | При наличии всех токенов — до 85% |

### 2.4 Зависимости от токенов и окружения

| Ресурс | Статус | Влияние |
|--------|--------|---------|
| VK_SERVICE_TOKEN | ✅ Установлен | VK-поиск работает |
| VK_USER_TOKEN | ❌ Не установлен | Стены, друзья, фото недоступны |
| TELEGRAM_API_ID/HASH/PHONE | ✅ Установлены | Методы A и B работают |
| Telethon session | ❌ Не создана | Метод C (поиск по каталогу) недоступен |
| SEARCH4FACES_API_KEY | ❌ Не установлен | Поиск по фото через Playwright (медленный) |
| ANTHROPIC_API_KEY | ❌ Не установлен | AI-саммари недоступны (пайплайн работает) |
| Snoop/Maigret/Sherlock | ❌ Не установлены | Поиск по username недоступен |
| Российский IP | ❌ Нет | ФССП API, Росфинмониторинг, Госуслуги/Сбер oracle |

### 2.5 Скорость

| Сценарий | Время | Примечание |
|----------|-------|------------|
| Минимальные данные (только ИНН) | 5-8 мин | Мало источников для обогащения |
| Полные данные (ИНН + ФИО + ДР + город) | 10-15 мин | Все этапы активны |
| С Holehe (Stage 4) | +25-30 сек | На каждый email |
| С Playwright-скрейпингом | +15-30 сек | На каждый сайт |
| **Целевое время** | **3-5 мин** | После оптимизации параллельности |

## 3. Конкурентная позиция

### 3.1 СЛЕД ВПЕРЕДИ Буратино

| Преимущество | Описание |
|-------------|----------|
| **Бесплатность** | 0 руб. vs 3,000-50,000 руб./мес. Радикальное преимущество для малых команд |
| **Open source потенциал** | Код на GitHub, возможность кастомизации и аудита |
| **Поведенческий анализ** | Анализ текстов VK-постов, выделение тем, тональности — у Буратино нет |
| **Социальный граф** | vis.js + NetworkX + Louvain community detection — визуальный анализ связей |
| **Гео-карта** | Leaflet.js + тепловая карта по 98 городам из VK-постов |
| **Username-поиск** | 8,700+ сайтов (Snoop 5,372 + Maigret 3,000+ + Sherlock 400+) |
| **Forgot-password oracle** | 8 сервисов: утечка масок email/телефона через формы восстановления пароля |
| **Маркетплейс-сканер** | 6 площадок: Avito, Youla, CIAN, Auto.ru, Яндекс, VK Маркет |
| **9-категорийный скоринг** | Структурированная оценка рисков с radar-чартом и пояснениями |
| **INN-first архитектура** | Верификация личности через ФНС до начала поиска — минимизация ложных срабатываний |
| **Telethon-интеграция** | Прямой поиск по каталогу Telegram (не только t.me проверка) |
| **Real-time progress** | Подробный прогресс-бар с логами по каждому этапу |
| **Precise mode** | Human-in-the-loop: пауза после Stage 3 для подтверждения профилей |
| **Self-hosted** | Данные остаются на локальной машине, нет облачного оператора |
| **YaSeeker** | Яндекс-разведка: Коллекции, Дзен, Музыка |
| **Activity timeline** | Временной анализ активности по VK-постам |
| **Risk radar chart** | Визуализация рисков по 9 категориям (Chart.js) |

| Банкротство | ✅ ЕФРСБ | ✅ ЕФРСБ |

## 8. Стратегическая оценка

### 8.1 Текущая позиция

СЛЕД сегодня — это **60-65% от функциональности Буратино по ширине покрытия**, но с **уникальными аналитическими возможностями**, которых нет ни у одного конкурента. Платформа находится в состоянии «рабочий прототип с пробелами»: ядро (INN-идентификация, VK-поиск, санкции, скоринг, отчеты) работает стабильно, но периферийные источники (ФССП, фото, username tools) требуют настройки или доработки.

### 8.2 Непреодолимые разрывы

Некоторые функции Буратино **невозможно реализовать** в рамках бесплатного open-source проекта:

| Функция | Причина | Альтернатива в СЛЕД |
|---------|---------|---------------------|
| Телефон -> ФИО (телеком) | Данные операторов доступны только по лицензии СОРМ или партнерству. Стоимость: от 500,000 руб./год | Телефон -> Telegram -> имя (бесплатно, но частичное покрытие). Телефон -> утечки (имена из breach DBs) |
| Адрес/прописка | Закрытая база ФМС/МВД | Гео-экстракция из VK-постов (косвенно) |
| Кредитная история | Платные бюро (Equifax, НБКИ) | Анализ долгов через ФССП/банкротство |
| CAPTCHA bypass (системный) | Требует anti-captcha сервисов или партнерств | Anti-captcha API ($3/1000 запросов) как частичное решение |

### 8.3 Достижимая конкурентная позиция

После выполнения 90-дневного плана СЛЕД может достичь следующей позиции:

### 8.4 Целевое позиционирование

```
Буратино = "Швейцарский нож за деньги"
  - Широко, быстро, просто
  - Но: дорого, закрыто, данные у третьих лиц, нет аналитики

СЛЕД = "Аналитическая лаборатория для глубоких проверок"
  - Глубже, умнее, бесплатно, приватно
  - Но: медленнее, требует настройки, нет телеком-данных
```

**СЛЕД не должен пытаться стать клоном Буратино.** Правильная стратегия:

1. **Закрыть критические пробелы** (телефон как input, токены, сломанные источники) — чтобы не было «пустых мест» в отчете
2. **Углублять уникальные преимущества** (поведенческий анализ, граф, гео, username search) — чтобы давать insights, которых нет ни у кого
3. **Выигрывать на стоимости и приватности** — бесплатное self-hosted решение для тех, кто не может / не хочет платить 50,000 руб./мес.

### 8.5 Целевая аудитория СЛЕД

| Сегмент | Почему СЛЕД | Почему не Буратино |
|---------|------------|-------------------|
| Малый бизнес (HR/СБ) | Бесплатно, достаточно глубоко | Буратино слишком дорого |
| OSINT-исследователи | Open source, кастомизация, глубокая аналитика | Буратино — черный ящик |
| Журналисты-расследователи | Self-hosted, конфиденциальность, бесплатно | Буратино отслеживает запросы |
| Фрилансеры (due diligence) | Бесплатно, визуальные отчеты | Буратино — подписка |
| Крупные СБ | Кастомизация, self-hosted, API | (Для них Буратино тоже подходит) |

### 8.6 Риски

| Риск | Вероятность | Влияние | Митигация |
|------|-------------|---------|-----------|
| Блокировка VK API | Средняя | Высокое | Множественные fallback-стратегии уже есть |
| Закрытие публичных реестров | Низкая | Высокое | Кеширование, альтернативные источники |
| Усиление CAPTCHA повсеместно | Высокая | Среднее | Anti-captcha бюджет, API-токены |
| Изменения в законодательстве (152-ФЗ) | Средняя | Высокое | Self-hosted снижает юридические риски |
| Блокировка breach APIs | Средняя | Среднее | Множественные API, локальная LeakDB |

### 8.7 Финальный вердикт

**СЛЕД уже сегодня превосходит Буратино в аналитической глубине** — поведенческий анализ, социальный граф, гео-карта, risk radar, username-поиск на 8,700+ сайтах. Этого нет ни у кого на рынке.

**Главная слабость — ширина покрытия и скорость.** Отсутствие телефона как входного параметра, нерабочий ФССП, ненастроенные инструменты — это устранимые проблемы, решаемые за 25 часов (Месяц 1).

**Непреодолимый разрыв один**: телефон -> ФИО через телеком-данные. Это главная фича Буратино, и она недоступна бесплатно. Однако комбинация «телефон -> Telegram + утечки + forgot-password oracle» закрывает эту потребность на 40-50%.

**Реалистичный прогноз**: через 90 дней СЛЕД может стать **лучшим бесплатным OSINT-инструментом для проверки физических лиц на российском рынке**, превосходящим Буратино по глубине анализа и уступающим только по ширине покрытия госреестров и скорости. Для 80% задач due diligence этого будет достаточно.
