## 2. ФУНКЦИОНАЛ БУРАТИНО

### Подтверждённые возможности
- Поиск по открытым источникам (OSINT)
- Визуализация данных и связей
- Обработка: текст, фото, видео, аудио, геоинформация
- Консолидация из нескольких источников
- Поиск по городам (автодополнение на сайте)
- Выявление неочевидных связей между объектами

### Применение
- Проверка контрагентов и физлиц
- Кадровая безопасность
- Профилактика коррупции
- Выявление мошенничества
- Профилактика конфликта интересов

### Чего НЕ видно (потенциальные пробелы)
- **Нет API** (только веб-интерфейс)
- **Нет мобильного приложения**
- **Нет интеграции с ATS/HR-системами**
- **Нет автоматических отчётов за 3-5 минут** (инструмент для аналитика, не автоматизация)
- **Нет публичного прайса по проверкам** (только годовая лицензия)
- **Устаревший фронтенд** (jQuery, нет SPA)
- **Нет соцсети-специфичных функций** (VK поиск, Telegram discovery и т.д.)

### Технологии (наблюдаемые)
- Frontend: jQuery + плагины
- Аналитика: Яндекс.Метрика
- Упоминаемые OSINT-инструменты в обучении: Shodan, Google Dorks, Maltego, TheHarvester, Recon-Ng, SpiderFoot, Creepy
- Серверный стек: **не раскрыт**

## 6. СЛАБЫЕ МЕСТА БУРАТИНО (= наши возможности)

1. **Ручной инструмент** — требует обученного аналитика. СЛЕД автоматизирует 90% работы.
2. **Нет API** — нельзя встроить в существующие процессы.
3. **Устаревший UI** — jQuery-based, без SPA.
4. **Бизнес-модель = обучение** — софт вторичен. Мотивация развивать продукт ниже.
5. **ИП** — не ООО/АО. Для enterprise-клиентов это риск (один человек = single point of failure).
6. **Нет публичных отзывов** — не найдено ни одного отзыва в открытых источниках.
7. **Нет Telegram-канала** — отсутствует в поиске.
8. **Закрытое ценообразование Феликса** — 250k/год для расследований, но без публичных кейсов.
9. **Малая команда разработки** — один основатель-профессор + партнёры-преподаватели.

## 9. ВЫВОДЫ

### СЛЕД находится в уникальной нише
Ни один конкурент не предлагает **автоматизированный глубокий OSINT** с social graph, behavioral analysis, breach DB и risk scoring в одном пайплайне.

### Ближайшие шаги для усиления позиции
1. **API + интеграции** — откроют enterprise-сегмент
2. **Проверка документов** — базовое ожидание рынка
3. **Landing page + демо** — для привлечения клиентов
4. **Кейсы и метрики** — "нашли X скрытых рисков из Y проверок"
5. **Юридическая обёртка** — ООО, 152-ФЗ compliance, оферта
